This was one of the first Websites I ever created. Before Themes and CRA's they coded everything by hand. This Project was to create responsive webpage using nothing but your own HTML and CSS

Ymir Boat Club is a local band so I decided to make a little webpage for them. With the skills I have now I could and should improve this page immensly, but it's a nice reminder of how much work used to go into websites, and how much work to takes to handcode a website from scratch
