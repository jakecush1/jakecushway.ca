Hours Logger

I created the first version of this application when working on my computer and wanting to log hours for different jobs and projects. I used HTML, CSS, Javascript and local storage to store the data.

I created the second version of this application when I realized it would be beneficial to be able to access the application from any machine and still be able to log hours on the same database. I used HTML, Css and Javascript in the front-end, and Google Firebase in the backend. Google Firebase is a free cloud storage database that is easy to learn and very versitile. The benefit of using a cloud storage versus a local storage is that the data can be accessed on any machine.
