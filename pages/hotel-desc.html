Hotel Reservation Website

This is a fully functional hotel reservation website. I first created the architechture and planning for the database, then using PHP and MYSQL made it work.

How it Works:
First you create an account and login. The data is saved in the MYSQL database on the server. Once you have an account you can make reservations if the room is available. The program then assigns you an access key and a room number and creates a reservation under your account name. You can also edit and delete your reservations which will change the room availability back to available, and return the assigned key.

There is also Admin or Manager functionalility where you can view all the current hotel guests, and the reservations. You can login to the admin portal by clicking on Administration link in the footer, and logging in with the credentials Manager and password
